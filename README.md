# Linux-VPN-and-Secure-File-Sharing
System Administration laboratory project implementing OpenVPN-based secure communication and Samba file sharing over a VPN between two Ubuntu 22.04 virtual machines.


# VPN Setup and Samba Access

# Project Overview
This project demonstrates a secure OpenVPN tunnel between two Ubuntu 22.04 virtual machines
(SriLanka-Client and US-Server) and secure Samba file sharing over the VPN.

# Technologies Used
- Ubuntu 22.04 LTS
- OpenVPN + Easy-RSA
- Samba
- Oracle VirtualBox

# Repository Structure
- `report/` – Full lab report
- `configs/` – OpenVPN, Samba, Netplan configs
- `outputs/` – Command outputs
- `screenshots/` – Proof screenshots
- `appendix/` – Commands used

# Verification
- VPN connectivity verified with 0% packet loss
- Samba accessed securely over VPN
